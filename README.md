# Wind Dynamics

A NeoForge 1.21.1 addon starter for Create Aeronautics.

## Included
- world-scale wind map cache
- weather-aware wind sampling
- directional wind sensor with one-side redstone output
- sail hub placeholder for later Create / Aeronautics integration

## Build
- `./gradlew runClient`
- `./gradlew build`

## Next step
